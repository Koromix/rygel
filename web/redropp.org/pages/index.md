# Overview

Redropp is a **file transfer web application** featuring:

- Support for internal users and SSO accounts
- End-to-end encryption, with transparent download decryption (with a service worker)
- Age-compatible encryption format for interoperability
- Encrypted file fragments are stored on an S3 backend
- Configurable global per-user storage quota
