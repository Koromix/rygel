  • Rich features
  • Quickly implemented and test
  • Free and open-source

This is the official website of Goupile, a free and open-source tool for creating forms on the fly.

  • Quickly implemented
  • Rich features
  • No advanced computer skills needed
  • Integrates with data warehousing solutions

Goupile is composed of five main modules :

  • a form editor defines the architecture, configuration, user administration
  • a page editor codes the widgets (label, type, options), the sections of the forms
  • a form overview displays the visual control
  • a data overview and export module displays the data and provide export function (in csv or xlsx formats)
  • a dashboard displays number of records, number of complete cases
