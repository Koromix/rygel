Code repository

Goupile is a free and open-source tool, licensed under the MPL 2.0 license. You can freely download and use the goupile source code. Everyone is granted the right to copy, modify and redistribute it. Navigate and clone the code on our git repository.

Please note, however, that this is a read-only repository. Most of goupile development happens in a monorepository that contains other projects. A subset of this repository (with the relevant history) is regularly pushed to the read-only goupile repository.
