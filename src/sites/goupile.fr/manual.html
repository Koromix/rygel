Create your form

Form editor

The form editor allows defining the architecture, configuration and user administration. Here, you define the number of pages of the form, their order, and their styles.

Page editor

The page editor helps you develop the form. It is equivalent to a code editor with the difference that here you don't need advanced computer knowledge to develop your form: you can use predefined functions. These functions will generate a visual control in the form: a text field, a numeric field, a date field, a slider, etc.

Code with the editor. Immediatly check the result with the form overview.

During the development step, you can benefit from both editor and form overview. With the form editor, you can develop your form by combining the input fields in sequence. This will instantly generate the corresponding form on the form overview. At this step, you can already test your form and enter data.

Routine use

After the form has been finalized, the form may be used conjointly with the data overview module and the dashboard to check records and progress of data entry.

Dashboard

The dashboard displays the list of records with their status (completed, in progress, empty). From the dashboard, you can access to a specific record in order to complete it or check its values.

Data export

Test the form. Immediately check the result with the data overview.

The data overview module displays the data. It also provides an export function of the whole dataset. Currently, supported formats are csv and xlsx.
